tl-mobile-menu Component: A reponsive menu for mobile apps
Mobile Menu companion component is a good menu to use if you have few high-level menu items (less than 5) and your app is meant to be used on both desktop and mobile

Each menu item having child can be unwrapped (only 1 level allowed)

On desktop, mobile menu appears on the left
On mobile, mobile menu appears on the bottom, to ease navigation

To use it, just add this in you app root or layout component:
The following inputs can be provided:
  • menuStyle: may be 'shadowed' or 'gradient'
  • iconSize: choose menu item icon size, in ox
