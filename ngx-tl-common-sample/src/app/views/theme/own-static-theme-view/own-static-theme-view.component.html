To build you own static style to use with ngx-tl-common, you just need to implement the theme variable in the :host pseudo-class within a global stylesheet of your app:
  • Within the root style.(s)ccs of your application
  • In an other ot dedicated CSS file. in this case, reference it as global stylesheet in your application angular.json file
