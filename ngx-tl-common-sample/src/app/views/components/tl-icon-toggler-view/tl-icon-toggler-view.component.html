Inputs
Name Type Description Default Value
size number Size of the square icon, in px 40
initialState boolean Initial state of the icon toggler
Use of an input property setter to handle initial value change if synchrone state is set		 false
color string Accentuation color of the icon when toggled on 'tl-outline'
icon string Icon, in the tl-icon-system 'tl-ion-heart'
iconOn string Icon, in the tl-icon-system, when the toggler is toggled on
If undefined, icon is the default 'icon'		 undefined
bouncingRatio number Bounding ratio
1.8 by default		 1.8
synchrone boolean Tells if component is synchrone, which means reacts to any inital state change false
unclickable boolean Prevents click event and action false
Events
Name Payload Type Description
toggleOn any Event that is raised when toggler is toggled on
toggleOff any Event that is raised when toggler is toggled off
toggle boolean Event that is raised when toggler is toggled
Carries the new state as payload
