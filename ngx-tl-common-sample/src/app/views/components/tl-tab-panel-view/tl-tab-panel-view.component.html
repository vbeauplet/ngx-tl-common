Inputs
Name Type Description Default Value
title string Title of the tab ''
imageUrl string Tab favicon URL
Let empty if none		 ''
icon string Tab icon
Let empty if none		 ''
alignment string Alignment of the root div of the tab content (div are on columns by default) 'left'
size string Children tabs
Size of the card, can be tl-big, tl-medium, tl-small.. Medium by default		 'tl-medium'
tlStyle string Tl Style of the card 'tl-neumorphic'
fixedHeight number Fixed height, in px. If not provided, height is auto -1
padding string Padding, in the tl-padding system
May be regular, small, big, xbig...		 'regular'
selectedTab string Selected tab ''
hasCloseTabButtons boolean Tells if ""close tab"" buttons shall be proposed true
hasAddTabButton boolean Tells if ""add tab"" buttons shall be proposed true
shrinkableTabs boolean Tells if tabs shall be srinked (title ellipsed) in case too many tabs for the width
If false, horizontal scroll is possible
True by default
Warning: cannot be set to false if style is neumorphic		 true
Events
Name Payload Type Description
closeTab string Event which is emitted when user click on a close button of a particular tab
Carried payload is the tab title
addTab any Event which is emitted when user click on the "add tab" button
