Inputs
Name Type Description Default Value
tlContainerNature boolean Tells if eleemnt is a container flex block or a flex-block
Flase by defualt, which means it is not a container by default		 false
tlStyle string Tl Style of the element
Has effect only if component is a container (see tlContainerNature flag)		 'tl-neumorphic'
direction string Direction of the action center, may be 'row' or 'column' 'row'
buttonStyle string Style of the buttons, neumorphic by default 'tl-neumorphic-sharp'
buttonShape string Shape of the buttons, round by default 'round'
buttonSize number Size of the inner buttons, in px 50
buttonMargin number Size of the inner buttons, in px 10
buttons ITlNamedIcon[] Button Definitions []
labelAsPopover boolean Tells if provided labels shall be displayed as popovers
False by default		 false
loadingStatus number Loading status of the action center -1
Events
Name Payload Type Description
clickButton string Events to be emitted when a button of the action center is clicked
Carried payload is the name of the button icon
