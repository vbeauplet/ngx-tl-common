Name Type Description Default Value
status number Loading status of the button -1
icon string Button literal icon ''
shape string Button shape: 'round' or 'square'. Default value is 'round' 'round'
tlStyle string Style of the pushable button
tl-neumorphic-sharp by default		 'tl-neumorphic-sharp'
label string Label of the icon, if any
Can be let undocumented if none		 ''
size number Default size of the icon button, in px
50 by default		 50
margin number Margin of the button, in px 10
labelPosition string Label position, if any
May be 'right' of 'bottom'
Bottom by default		 'bottom'
color string Icon color when status is -1 (for stateful buttons, always otherwise)
Possible values are "tl-main", "tl-soft", 'tl-failure', "tl -neutral", "tl-success", "tl-outline", ...
"main" by default		 'tl-main'
disabled boolean Tells if the button is disabled (cannot be clicked)
False by default		 false
