Inputs
Name Type Description Default Value
size string Size of the String Picker container block.
Shall fit tl-size system
tl-full by default		 'tl-full'
fontSize number Font size, in px
10 by default		 16
elementMargin number Margin between each string picker lines, in px 6
alignment string Alignment of the proposals. May be 'left' or 'center' (default) 'center'
singleSelection boolean Tells if only one string can be selected
False by default		 false
forceSelectOne boolean Only matters if the string picker runs in single selection mode
If this option is set to true and single selection too, selcted i tem cannot be unselected
At any time, there will be only one selected item
True by default (but single selection is set to false by def ault)		 true
initialItems string[] List of all initial items among which to select
Mandatory, empty by default		 []
initialSelectedItems string[] List of initially selected items []
synchrone boolean Synchronize selected items with initial selected items input
It means if true anytime table provided as initialSelectedItems chan ges,
table of component selected item is updated accordingly.
If false, initialSelectedItems are copied only once at component initiali zation and compoennt lives its life afterward
False by default		 false
editable boolean Tells if list of proposed items is editable, which means items can be created via an input field false
inputId string ID of the item creation input.
Shall be documented if 2 or more editable string-pickers are on the same page		 'create-item-input-id'
displaySpinner boolean Display spinner flag true
Events
Name Payload Type Description
clickItem string Event that is raised when an item is clicked
String that is clicked is carried by the event
selectItems string[] Event that is raised when selection is changed
List of all selected items is carried by this event
selectItem string Event that is raised when an item is selected
Selected item is carried by this event
unselectItem string Event that is raised when an item is unselected
Unselected item is carried by this event
createItem string Event that is raised when an item is created
