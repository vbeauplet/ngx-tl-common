Inputs
Name Type Description Default Value
size string Size of the Color Picker container block. tl-full by default 'tl-full'
colorCircleSize number Size of the color items circles, in px
50 by default		 50
elementMargin number Margin between each color items of the color picker, in px 6
alignment string Alignment of the proposals. May be 'left' or 'center' (default) 'center'
singleSelection boolean Tells if only one color item can be selected
True by default		 true
forceSelectOne boolean Only matters if the color picker runs in single selection mode
If this option is set to true and single selection too, selcted it em cannot be unselected
At any time, there will be only one selected item
False by default		 true
initialItems ColorItem[] List of all initial color items among which to select
Mandatory, empty by default		 []
initialSelectedItems ColorItem[] List of initially selected color items []
synchrone boolean Synchronize selected items with initial selected items input
It means if true anytime table provided as initialSelectedItems chan ges,
table of component selected item is updated accordingly.
If false, initialSelectedItems are copied only once at component initiali zation and compoennt lives its life afterward
False by default		 false
Events
Name Payload Type Description
clickItem ColorItem Event that is raised when a color item is clicked
Color item that is clicked is carried by the event
selectItems ColorItem[] Event that is raised when selection is changed
List of all selected items is carried by this event
selectItem ColorItem Event that is raised when an item is selected
Selected item is carried by this event
unselectItem ColorItem Event that is raised when an item is unselected
Unselected item is carried by this event
