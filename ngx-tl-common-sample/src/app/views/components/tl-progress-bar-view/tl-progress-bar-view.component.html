size string Size of the component 'tl-small'

30 by default		 30
color string Color of the inner progressing bar, in the tl-common system
tl-success by default		 'tl-success'
tlStyle string tl-common style og the progress bar
Neumorphic inset by default		 'tl-neumorphic-inset-sharp'
total number Total numbe of progress states 1
current number Current progress bar state 0
title string Title, if any
Let emtpty if none		 ''
showProgressAsText boolean Tells if step progress shall also be shown in text mode (as current / total)
True by default		 true
timeMode boolean Tells if this is a progress bar to display time progression
In this case, total and current inputs shall be given in ms		 false
Events
Name Payload Type Description
clickItem ColorItem Event that is raised when a color item is clicked
Color item that is clicked is carried by the event
selectItems ColorItem[] Event that is raised when selection is changed
List of all selected items is carried by this event
selectItem ColorItem Event that is raised when an item is selected
Selected item is carried by this event
unselectItem ColorItem Event that is raised when an item is unselected
Unselected item is carried by this event
