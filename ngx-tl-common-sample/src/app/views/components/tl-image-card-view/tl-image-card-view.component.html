Example
A hedgehog is any of the spiny mammals of the subfamily Erinaceinae, in the eulipotyphlan family Erinaceidae.

There are seventeen species of hedgehog in five genera found through parts of Europe, Asia, and Africa, and in New Zealand by introduction.
Inputs
Name Type Description Default Value
size string Size of the card, can be tl-big, tl-medium, tl-small.. Medium by default 'tl-medium'
tlStyle string Tl Style of the card 'tl-neumorphic'
fixedHeight number Fixed height, in px. If not provided, height is auto -1
isLoading boolean Tells if card is loading and shall therefore display a loader as content false
margined boolean Tells if card is margined, true by default true
title string Title of the card, if any
Empty string if one (default)		 ''
actionCenterStyle string Style of the action center that contains the active icons 'tl-soft-transparent'
imageSrc string URL of the image or photo to display at the top of the card ''
imageShape string Shape of the photo or image 'square'
imagePadding number Image padding, in px 0
imageIcons ITlNamedIcon[] Active icons to propose to the user in card image or phot
If none no 'action center' will be displayed		 []
bodyPadding number Body padding, in px
20px by default		 20
Events
Name Type Description Default Value
clickButton string Event to raise when clicking on an active icon
Payload is te name of the named icon
