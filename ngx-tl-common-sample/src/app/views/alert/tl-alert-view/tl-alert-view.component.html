You can tweak the alert component using the following inputs:
Name Type Description Default Value
tlStyle string Style of the alerts, from tl-style system
Tl-neumorphic-popover by default		 'tl-neumorphic-popover'
buttonStyle string Style of the decision buttons, from the tl-style system
tl-neumorphic by default		 'tl-neumorphic'
canClose boolean Tells if a close button shall be visible for non-decisive popups true
size string Size of the alerts 'tl-big'
borderRadius string Border radius of the alerts, in the tl-border-radius system 'tl-big-br'
hideNonDecisivePopups boolean Tells if non-decisive alerts (without decision) shall be hidden
False by default		 false
